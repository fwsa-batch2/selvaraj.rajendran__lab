## **RUBY**


> print vs puts

While the print method allows you to print information in the same line even multiple times, the puts method adds a new line at the end of the object.

```
print "hello world"
puts "hello world"
```
 * output
 ```
 hello worldhello world
 ```

 >difference
 
  ```
puts "hello world"
print "hello world"
 ```
 * output
 ```
 hello world
hello world
 ```

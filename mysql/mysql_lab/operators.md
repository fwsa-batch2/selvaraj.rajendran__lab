 ## operators

| Operator 	| Description               |
|:---------:|:-------------------------:|
|  =        |Equal to                   | 
|  >        |Greater than               |
| >=        |Greater than or equal to   |
| < 	    |Less than                  |
| <= 	    |Less than or equal to      |
| != 	    |Not equal to               |
|  <>       |	Not equal to            |
|  <=>  	|NULL-safe equal to operator|
